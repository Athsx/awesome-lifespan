__mTor__ (mammalian target of rapamycin) | 哺乳动物雷帕霉素靶蛋白| 是细胞生长和增殖的重要调节因子。大量研究显示mTOR信号途径调控异常与细胞增殖密切相关. mTOR属于一个重要的真核细胞信号，其稳定性影响T细胞中细胞因子的表达，参与免疫抑制，影响转录和蛋白质合成，调节细胞的生长、凋亡、自噬等。mTOR则被认定为肿瘤治疗的新靶点，对运动、代谢、神经等疾病具有重要的调节作用。
